# Dog API testing 
Implemented with Javas jayway RestAssured + TestNG and Apache Maven build management tool

#How to Use
Go to APItask1 class in /com.org.api/src/test/java/CIBQAssement/com/org/api/APItask1.java
Run as TestNG to execute all 4 testcases that answer he 4 questions
Methods:
TC101_AllDogBreeds()  	- returns all dog breeds
TC102_VerifyRetreiver() - verifies that retriever exist
TC103_RetreiverList() 	- returns all sub breeds under retreiver
TC104_GoldenRandomImg() - returns a random retriever golden image

results are returned on console window
